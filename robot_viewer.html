📹 ESP32-CAM Stream

Status: Not connected

📋 Quick Start

  1. Enter your ESP32-CAM IP address (check Serial Monitor)
  2. Click "Connect Stream"
  3. Run the hand control client in a separate terminal
  4. Control the robot with your hands while watching the stream!

Hand Control Command:

python -m client_hand_control.main --server ws://YOUR_SERVER:8080/control --token test123 --camera 0 --preview
